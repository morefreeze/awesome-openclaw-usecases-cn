# 基于电话的个人助理

## 痛点

你想从任何电话访问你的AI代理，而不需要智能手机应用或互联网浏览器。你需要在驾驶、行走或双手被占用时的免提语音辅助。

## 功能介绍

ClawdTalk使OpenClaw能够接听和拨打电话，将任何电话变成你的AI助理的网关。你可以：
- 拨打电话号码通过语音与你的AI代理交谈
- 通过语音获取日历提醒、Jira更新和网页搜索结果
- 与Telnyx集成以实现可靠的电话连接

短信支持即将推出。

## 提示词

```
你可以通过电话联系到。当我打电话时，问候我并询问我需要什么帮助。

对于日历查询："我今天的日程是什么？"
对于Jira更新："显示我的未处理工单"
对于网页搜索："搜索AI代理的最新消息"
```

## 所需技能

- [ClawdTalk](https://github.com/team-telnyx/clawdtalk-client)
- 日历技能(Google Calendar或Outlook)
- Jira技能
- 网页搜索技能
