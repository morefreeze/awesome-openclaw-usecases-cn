# 定制早间简报

你醒来后,每天的前30分钟都在追赶信息——滚动新闻、查看日历、回顾待办事项,试图弄清楚今天重要的事情。如果所有这些都已经完成,并作为短信等在你面前呢?

这个工作流让OpenClaw在每天的预定时间向你发送完全定制的早间简报,涵盖新闻、任务、想法和前瞻性建议。

## 功能介绍

- 每天同一时间(例如,早上8:00)通过Telegram、Discord或iMessage发送结构化的早间报告
- 通过浏览网页研究与你兴趣相关的隔夜新闻
- 查看你的待办事项列表,列出当天的任务
- 在你睡觉时生成创意输出(完整的脚本、电子邮件草稿、商业提案——不仅仅是想法)
- 推荐AI可以自主完成的任务,以帮助你度过这一天

## 痛点

你把最有生产力的早晨时间都花在了熟悉情况上。与此同时,你的AI代理整夜都在闲置。早间简报将闲置的夜间时间变成了富有成效的准备时间——你醒来时工作已经完成。

## 所需技能

- Telegram、Discord或iMessage集成
- Todoist / Apple提醒 / Asana集成(无论你用哪个来管理任务)
- [x-research-v2](https://clawhub.ai)用于社交媒体趋势研究(可选)

## 设置方法

1. 将OpenClaw连接到你的消息平台和任务管理器。

2. 提示OpenClaw:
```text
我想设置一个定期的早间简报。每天早上8:00,
通过Telegram给我发送一份报告。

我希望这份报告包括:
1. 与我兴趣相关的新闻故事(AI、初创企业、科技)
2. 我今天可以创作的内容创意
3. 我今天需要完成的任务(从我的待办事项列表中提取)
4. 你今天可以为我完成的任务建议

对于内容创意,请写出完整的草稿脚本/大纲——不仅仅是标题。
```

3. OpenClaw会自动安排这个任务。第二天早上查看你的消息以验证它是否正常工作。

4. 随着时间的推移进行自定义——只需给你的机器人发短信:
```text
在我的早间简报中添加天气预报。
停止包含一般新闻,只关注AI。
每天早上包含一句励志名言。
```

5. 如果你想不出要包含什么,你不必费心——只需说:
```text
我希望这份报告包含与我相关的内容。
考虑一下在这份报告中加入什么最有帮助。
```

## 关键见解

- AI推荐的任务部分是最强大的部分——它让代理主动思考如何帮助你,而不是等待指令。
- 你只需发短信就可以定制简报。说"在我的早间简报中添加股票价格",它就会更新。
- 完整的草稿(不仅仅是想法)是节省时间的关键。醒来时看到的是脚本,而不是建议。
- 无论你身处哪个行业——包含任务、新闻和前瞻性建议的早间简报都普遍有用。
