# 第二大脑

你会产生想法、找到有趣的链接、听到要读的书——但你从来没有一个好的系统来捕捉它们。Notion变得复杂，Apple Notes变成了10000条未读条目的坟场。你需要像给朋友发短信一样简单的东西。

这个工作流将OpenClaw变成一个记忆捕捉系统，你可以通过短信与之交互，后端是一个你可以随时浏览的可搜索自定义UI。

## 功能介绍

- 通过Telegram、iMessage或Discord向你的OpenClaw发送任何内容——"提醒我读一本关于本地LLM的书"——它会立即记住
- OpenClaw的内置记忆系统永久存储你告诉它的所有内容
- 自定义Next.js仪表板让你搜索所有记忆、对话和笔记
- 全局搜索(Cmd+K)跨所有记忆、文档和任务
- 没有文件夹，没有标签，没有复杂的组织——只有文本和搜索

## 痛点

每个笔记应用最终都变成了一项苦差事。你停止使用它，因为组织的摩擦比忘记的摩擦更高。关键见解是：**捕捉应该像发短信一样容易，检索应该像搜索一样容易**。

## 所需技能

- Telegram、iMessage或Discord集成(基于文本的捕捉)
- Next.js(OpenClaw会为你构建——不需要编码)

## 设置方法

1. 确保你的OpenClaw连接到你喜欢的消息平台(Telegram、Discord等)。

2. 立即开始使用——只需给你的机器人发送任何你想记住的内容：
```text
嘿，提醒我读《设计数据密集型应用》
保存这个链接：https://example.com/interesting-article
记住：约翰推荐了第五街的那家餐厅
```

3. 通过提示OpenClaw构建可搜索UI：
```text
我想构建一个第二大脑系统，我可以在其中查看我们所有的笔记、
对话和记忆。请用Next.js构建。

包括：
- 所有记忆和对话的可搜索列表
- 全局搜索(Cmd+K)跨所有内容
- 按日期和类型过滤的能力
- 干净、极简的UI
```

4. OpenClaw会为你构建并部署整个Next.js应用。导航到它提供的URL，你就有了自己的第二大脑仪表板。

5. 从现在开始，无论何时你想到什么——在路上、在会议中、睡前——只需给你的机器人发短信。需要找东西时回到仪表板。

## 关键见解

- 力量在于**零摩擦捕捉**。你不需要打开应用、选择文件夹或添加标签。只需发短信。
- OpenClaw的记忆系统是累积的——它记住你告诉过它的**一切**，随着时间的推移变得更强大。
- 你可以从手机给机器人发短信，它会在你的电脑上构建东西。界面就是对话。
