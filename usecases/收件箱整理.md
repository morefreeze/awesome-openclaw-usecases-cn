# 收件箱整理

时事通讯会占据你的收件箱，往往堆积如山，根本没被打开过。

## 所需技能
[Gmail OAuth 设置](https://clawhub.ai/kai-jar/gmail-oauth)。

## 设置方法
1. [可选] 专门为 OpenClaw 创建一个新的 gmail 账户。
2. [可选] 从你的主邮箱退订所有时事通讯，然后使用 OpenClaw 邮箱重新订阅。
3. 安装技能并确保其正常工作。
4. 指示 OpenClaw：
```txt
我希望你每天晚上8点运行一个 cron 作业，读取过去24小时内的所有时事通讯邮件，为我提供最重要内容的摘要以及阅读更多的链接。然后询问我对摘要内容的反馈，并根据我的偏好更新记忆，以便在未来的任务中提供更好的摘要。
```
