# 多代理内容工厂

你是一名内容创作者,同时兼顾研究、写作和跨平台设计。从寻找热门话题、撰写脚本到生成缩略图,每一步都占用你数小时的时间。如果有一个由专业代理组成的团队在一夜之间完成所有这些工作呢?

这个工作流在Discord中设置了一个多代理内容工厂,不同的代理在专门的频道中处理研究、写作和视觉资产。

## 功能介绍

- **研究代理**每天早上扫描热门故事、竞争对手内容和社交媒体,寻找最佳内容机会
- **写作代理**获取最佳创意并撰写完整的脚本、帖子或时事通讯草稿
- **缩略图代理**为内容生成AI缩略图或封面图片
- 每个代理在自己的Discord频道中工作,保持一切井井有条且可审查
- 按计划自动运行(例如,每天早上8点),让你醒来时就能看到完成的内容

## 痛点

内容创作有三个阶段——研究、写作和设计——大多数创作者都是手动完成这三个阶段。即使使用AI写作工具,你仍然必须逐个提示它们。这个系统将代理链接在一个流水线中,一个代理的输出作为下一个代理的输入,完全无需人工干预。

## 所需技能

- 与多个频道的Discord集成
- `sessions_spawn` / `sessions_send` 用于多代理编排
- [x-research-v2](https://clawhub.ai)或类似工具用于社交媒体研究
- 本地图像生成(如Nano Banana)或图像生成API
- [knowledge-base](https://clawhub.ai)技能(可选,用于基于RAG的研究)

## 设置方法

1. 设置一个Discord服务器(或让OpenClaw为你设置——只需说"为我们设置一个Discord")。

2. 为每个代理创建频道:
   - `#research` — 热门话题和内容机会
   - `#scripts` — 书面草稿和大纲
   - `#thumbnails` — 生成的图像和封面艺术

3. 提示OpenClaw:
```text
我想让你在Discord中为我构建一个内容工厂。
为不同的代理设置频道:

1. 研究代理(#research): 每天早上8点,研究我的细分领域中最热门的
   故事、竞争对手内容以及社交媒体上表现良好的内容。发布前5大
   内容机会并附上来源。

2. 写作代理(#scripts): 获取研究代理的最佳创意,
   撰写完整的脚本/帖子/时事通讯草稿。发布在#scripts中。

3. 缩略图代理(#thumbnails): 为内容生成AI缩略图或封面图片。
   发布在#thumbnails中。

让他们的所有工作在不同的频道中组织起来。
每天早上自动运行这个流水线。
```

4. 根据你的平台进行自定义:
```text
我专注于X/Twitter帖子,而不是YouTube。将写作代理
更改为生成推文帖子而不是视频脚本。
```

## 关键见解

- 强大之处在于**链式代理**——研究为写作提供素材,写作为缩略图提供素材。你不需要单独提示每个步骤。
- Discord频道让你可以轻松地分别查看每个代理的工作并提供反馈,例如"脚本太长"或"更多关注AI新闻"。
- 你可以将其调整为任何内容格式:推文、时事通讯、LinkedIn帖子、播客大纲、博客文章。
- 使用本地模型进行图像生成(如Mac Studio上的Nano Banana)可以降低成本并让你获得更多控制权。
